# BlockchainAPI

This api help to add data to ethereum blockchain and verify the data from the blockchain

# Install NPM
```
npm install
```
# Install Truffle
```
npm i -g truffle
```
# Set Execution Policy
```
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
```
# Truffle Build
```
truffle build
```
# Truffle Depoly
```
truffle depoly
```
# Start NPM
```
npm start
```
